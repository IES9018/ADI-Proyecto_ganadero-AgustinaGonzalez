# Flujo UX – Sistema Ganadapp

## 1. Objetivo

Definir la estructura de navegación y experiencia de usuario del sistema de gestión ganadera.


## 2. Principio de diseño

- simplicidad operativa para usuarios rurales
- reducción de carga cognitiva
- flujos cortos (máx. 3–4 pasos por tarea)
- prioridad a registro rápido de datos en campo


## 3. Estructura general de navegación

### Pantalla de inicio

- acceso a módulos principales:
  - ganado
  - eventos sanitarios
  - stock
  - trazabilidad
  - alertas


### Flujo 1: gestión de ganado

inicio → listado de animales → detalle → edición / creación


### Flujo 2: registro de evento sanitario

inicio → selección de animal → formulario de evento → confirmación


### Flujo 3: consulta de trazabilidad

inicio → búsqueda de animal → historial cronológico

### Flujo 4: gestión de stock

inicio → listado de insumos → actualización de cantidades → alertas automáticas

## 4. Comportamiento general

- persistencia automática de formularios
- validación inmediata de datos
- confirmación visual de acciones exitosas
