# Componentes UI – Ganadapp

## 1. Estructura general

La interfaz se organiza en componentes reutilizables para garantizar consistencia visual y funcional.


## 2. Componentes principales

### navbar principal

- acceso a módulos
- identificación de usuario
- logout


### card de animal

- información básica del animal
- estado sanitario
- accesos rápidos


### formulario de evento

- tipo de evento
- fecha
- descripción
- selección de animal


### tabla de stock

- insumo
- cantidad actual
- umbral mínimo
- indicador de alerta



### panel de alertas

- alertas sanitarias
- alertas de stock
- prioridad visual



## 3. Principios de diseño

- reutilización de componentes
- diseño responsive
- baja complejidad visual
