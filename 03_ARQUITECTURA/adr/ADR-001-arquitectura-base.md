# ADR-001: elección de arquitectura base

## Decisión
Se adopta una arquitectura modular monolítica basada en arquitectura hexagonal.

## Contexto
El sistema debe ser mantenible, educativo y extensible sin complejidad distribuida.

## Consecuencias
- simplicidad de despliegue
- menor complejidad operativa
- facilidad de aprendizaje
- limitación en escalado horizontal (aceptada en este contexto educativo)
