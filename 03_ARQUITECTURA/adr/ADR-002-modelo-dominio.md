# ADR-002: modelado de dominio

## Decisión
El dominio se centra en tres entidades principales:
- ganado
- eventos sanitarios/productivos
- stock de insumos

## Contexto
Se busca mantener el modelo simple pero representativo del negocio real.

## Consecuencias
- facilita trazabilidad
- reduce complejidad inicial
- permite evolución hacia módulos avanzados
