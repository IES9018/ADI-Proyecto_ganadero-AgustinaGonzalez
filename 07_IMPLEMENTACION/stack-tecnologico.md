# Stack tecnológico – Ganadapp

## 1. Objetivo

Definir las tecnologías base del sistema para garantizar coherencia arquitectónica y facilidad de implementación.


## 2. Backend

- Lenguaje: Python
- Framework: FastAPI
- ORM: SQLAlchemy (o equivalente ligero)
- Base de datos: SQLite (MVP) / PostgreSQL (evolución)
- Arquitectura: hexagonal (puertos y adaptadores)


## 3. Frontend

- HTML + CSS + JavaScript (MVP)
- Evolución posible: React o Vue
- Enfoque: interfaz simple y funcional


## 4. Base de datos

- relacional
- modelo normalizado
- soporte para migración futura

## 5. Herramientas

- Git + GitHub
- Docker (opcional futuro)
- PlantUML (diagramas)
- Draw.io (modelado visual)
